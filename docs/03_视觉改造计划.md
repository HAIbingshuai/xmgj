# 视觉问题清单与改造计划

> 每条按 问题描述 → 改造方案 → 实施 来推进

---

## 全局性问题（先修复，影响所有页面）

### G1. 颜色太素太灰，没有层次感
**问题：** 背景 `#F0F2F5`、卡片纯白、阴影 `0 1px 3px`，整体就像没做设计的 Bootstrap 默认样
**方案：** 增加色彩层次：背景加深、卡片用更柔和的圆角+阴影、主色在更多地方出现

### G2. 字体排版混乱
**问题：** 到处是 inline `style="font-size:0.85rem"`，没有统一的文字层级
**方案：** 给 CSS 增加 h1-h6 和 body/text-small/text-muted 的统一规范，去掉多余的 inline style

### G3. 卡片/面板没有视觉层次
**问题：** 左栏面板、右栏面板、表单卡片都是同一个样子，分不清主次
**方案：** 左栏用更浅的背景区分于右栏，表单卡片用更明显的投影

### G4. 按钮风格陈旧
**问题：** Bootstrap 默认的 `btn-primary` `btn-outline-secondary` 太通用
**方案：** 按钮增加圆角、hover 动效、图标+文字组合的一致性

### G5. 缺少微交互
**问题：** 点击、悬停、切换都没有过渡反馈
**方案：** CSS transition 覆盖所有可交互元素，增加点击波纹效果

### G6. 间距不一致
**问题：** 有的用 `px-3` `py-3`，有的用 `p-4`，有的 inline style 写 padding
**方案：** 统一间距规范：卡片 20px padding，区块间距 16px

## 各页面问题

### P1. 导航栏 — 太普通
- 普通的深蓝渐变，没有品牌感
- 当前页没有足够明显的激活态

### P2. 仪表盘 — 四个统计卡片像占位符
- 卡片只有纯色圆形图标+数字，没有设计感
- 整体太空，缺少视觉焦点

### P3. 事项列表 — 卡网格又平又挤
- 卡片无 hover 动效，像静态表格
- 筛选栏样式简陋
- 徽章颜色用 `yesno` 过滤器有 bug

### P4. ★ 事项详情 — 核心页面问题最多
- 紧凑 Header 太挤：← 返回 + 标题 + 徽章 + 描述在一行
- 左栏时间轴节点卡片设计粗糙，灰底白卡看着廉价
- 右栏步骤卡片展开后内容排版松散
- 结果区的绿色框/红色框样式扎眼
- 抽屉内容太素
- 按钮栏 `btn-act` 样式简陋
- 操作记录的时间线圆点太小

### P5. 表单页（事项/节点/步骤）— 三栏布局太空
- 左右两侧的提示卡片没什么实际作用
- 表单内部的 label 和 input 间距太大
- 提交按钮缺少视觉强调

---

## 改造原则

1. **先改 CSS（全局样式），再改模板**
2. **每条改动可独立验证**
3. **不改功能逻辑，只改视觉呈现**
4. **坚持改完一条再做下一条**

---

## 实施顺序

1. G1-G6：全局 CSS 翻新（style.css）
2. P1：导航栏重新设计
3. P2：仪表盘卡片重做
4. P3：事项列表翻新
5. P4：事项详情页重点改造（分多步）
6. P5：表单页精简美化
